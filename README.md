# Modern Portfolio Website

A clean, responsive portfolio website built with HTML, CSS, and JavaScript. Features a modern design with smooth animations and a user-friendly interface.

## Features

- Responsive design that works on all devices
- Smooth scroll animations
- Modern UI with subtle hover effects
- Contact form
- Social media integration
- Section-based navigation with active state
- Clean and well-commented code

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser
3. Customize the content in `index.html` with your information
4. Update the styling in `styles/main.css` if desired
5. Modify the JavaScript functionality in `js/main.js` as needed

## Customization

### Colors
The color scheme can be modified in the CSS variables at the top of `styles/main.css`:

```css
:root {
    --primary-color: #4070F4;
    --text-color: #2D2E32;
    --bg-color: #FFFFFF;
    --secondary-bg: #F9F9F9;
    --accent-color: #FFD700;
    --shadow-color: rgba(0, 0, 0, 0.1);
}
```

### Profile Image
Replace the placeholder image in the HTML file:
```html
<img src="path/to/your/image.jpg" alt="Profile Image" class="profile-img">
```

### Social Links
Update the social media links in the HTML file:
```html
<div class="social-links">
    <a href="your-linkedin-url" target="_blank"><i class="fab fa-linkedin"></i></a>
    <a href="your-behance-url" target="_blank"><i class="fab fa-behance"></i></a>
    <a href="your-github-url" target="_blank"><i class="fab fa-github"></i></a>
</div>
```
